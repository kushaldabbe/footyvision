# FootyVision

## Description
FootyVision is a Python project aimed at developing a comprehensive solution for processing and analyzing football (soccer) data.

## Setup
To set up the FootyVision project, clone the repository and install the required dependencies listed in `pyproject.toml`. Ensure that you're using Python 3.10 or higher.

## Quickstart
1. Clone the repository:
   ```bash
   git clone https://github.com/kushaldabbe/footyvision.git
   cd footyvision
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the project:
   ```bash
   python src/footyvision/cli.py
   ```

## Scope
This project will implement various modules for data ingestion, processing, and visualization.
